> 一些和朋友聊完的碎碎念

感觉自己的情绪就像是 Android 手机的内存一样，需要定期做一下内存清理，积累了太多负能量就容易 Stack Overflow.

自从 MBTI 火了之后，不仅仅是成为了一种社交筹码，对于我自己而言，虽然比起星座来说，稍微「科学」了那么一点。但在自己的潜意识里面，越来越把它当做一个 —— 对别人是一种刻板影响，对自己则成为了一种行为 SOP。好像我就应该按照自己的 MBTI 来处理一切的问题。

比如我自己的 MBTI，从 **ENFP -> ENTP -> INTP -> INTP** 一共演变了 4 次。以至于在今天处理很多问题时，都会把这些当成一种规训 / 借口。

比如，在面对情绪问题时，我会认为自己是个 T 人（带点优越感），不应该被这种「没有意义的」情绪给左右。实际上，是我自己的情绪处理能力低下。说的更简单一点，我很害怕、也不知道如何控制情绪。当情绪来临时，我下意识总会去压抑他。我总认为自己的理性是更加优势的能力，我擅长并习惯用自己的理性来「解构」问题，并且形成一个我认可的逻辑闭环。

然而，这样的方式也给了自己很多的「内存泄露」，刚开始运作的时候 Everything works like it should be. 但随着时间推移，我总感觉所谓的「逻辑闭环」永远不可达，今天我想通了 A -> B -> C，开心了舒服了。

第二天，我又会担心，是不是漏了个 D？或者，A -> B 合理吗？A 不可推导出 E 吗？

如此种种...

这种无限延伸的逻辑，堪比泰勒展开，完全停不下来，总认为自己需要想的更加明白、更透彻。

其实，我也知道，“道理我都懂”，有些问题不是你能想明白就想明白的。

今天和朋友聊了聊，他说最近他因为毕业的事情感到焦虑，所以寻求了几次心理咨询。

坦白而言，我对心理咨询一直是抱着，不是说「嗤之以鼻」的态度，而是我坚信：我需要逻辑的闭环，而不是情绪上的解构。

我总是会觉得，情绪这些问题都是虚无缥缈的，那些所谓的「情绪理论」，我和 GPT 也不是没有深夜交谈过，巴拉巴拉吐出来一堆，实际我能吸收的约等于 0。

我也是一个固执的人，不愿意尝试新的方法论，甚至很多时候带着很多偏见。所以这两个叠加在一起之后，就会让我觉得:

- 这都是啥啊，虚头巴脑的
- 听不懂，Get 不到，没有正反馈

另外，我是一个非常非常慢半拍的人，很多道理，想法，也许别人早就明白了，我需要很久，需要有非常 Solid 的体验 + 验证之后才能豁然开朗。

很感谢我的几个朋友，愿意在我大半夜 emo 的时候，听我唠唠叨叨讲之前已经讲过不下三遍的故事了。

他们很有耐心，也给了我很多建设性意见。也许是带着一些「我在麻烦他们」的情绪，也或者是他们确实说的很真诚。所以他们说的话，我很认真的听了，并且感觉吸收了不少。

聊完的那一刻，我很开心，开心是 —— 有朋友愿意真诚的接受我的负面能量；有朋友能给我中肯且有意义的观点。

差不多就这些吧，感慨一下。

**晚安**

export default BlogContainer

export async function getServerSideProps(context) {
  const { locale, query } = context
  const meta = await http.get('GetBlogMeta', {
    blogId: query?.key
  })
  time.setLocale(locale)
  return {
    props: {
      ...(await serverSideTranslations(locale, ['common', 'blog'])),
      meta: meta?.data,
    },
  }
}
